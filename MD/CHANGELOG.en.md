# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-10

### Added
- Initial release of BotanyPots Trees datapack
- Oak sapling recipe
- Birch sapling recipe
- Spruce sapling recipe
- Acacia sapling recipe
- Dark Oak sapling recipe
- Jungle sapling recipe
- Cherry sapling recipe
- Mangrove propagule recipe
- Complete documentation (README in English and Italian)
- MIT License
- .gitignore file

### Features
- All vanilla Minecraft tree types supported
- Balanced growth times (1200 ticks = 60 seconds)
- Automatic sapling and log drops
- Compatible with dirt-based soils

## [Unreleased]
